Determine minimal bootstrap console visibility
SolidgroundUX / sgnd-exe-common.sh / Minimal UI / _sgnd_minimal_say_should_print_console
Purpose
Determine whether a bootstrap message should be written before the full
SolidGroundUX UI layer is loaded.
Parameters
$1 Message type.
Globals (read)
SGND_LOG_LEVEL
Returns
0 if the message should be displayed.
1 if the message should be suppressed.
