Record a created file in the manifest
SolidgroundUX / create-workspace.sh / Manifest helpers / _manifest_record_file
Purpose
Append a file entry to the workspace manifest.
Arguments
$1 Absolute file path
Behavior
- Does nothing when WORKSPACE_MANIFEST is unset.
- Does nothing in dry-run mode.
- Appends a FILE record to the manifest.
Returns
0 always
