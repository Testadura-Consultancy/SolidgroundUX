Set or add a field line interactively
SolidgroundUX / metadata-editor-updated.sh / Main / _interactive_field_line_files
Purpose
Prompt for one '<Fieldname> : value' line and apply it to selected files.
Arguments
$1 APPLY_ALL
0 = first file only.
1 = all resolved files.
Returns
0 when all selected files were updated.
1 on failure.
2 when cancelled or no line was provided.
