List available deployment manifests
SolidgroundUX / deploy-workspace.sh / Manifests / _manifest_list
Purpose
List available manifests for the current source root.
Behavior
- Prints manifest filenames for the current SRC_ROOT namespace
- Sorted lexically
Inputs (globals):
SRC_ROOT
MANIFEST_BASE_DIR
Returns
0 on success
1 if manifest directory does not exist
Usage
_manifest_list
