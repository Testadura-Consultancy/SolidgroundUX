Discard the temporary deployment manifest
SolidgroundUX / deploy-workspace.sh / Manifests / _manifest_discard
Purpose
Remove the temporary manifest after an interrupted or failed deployment run.
Behavior
- Deletes MANIFEST_TMP if it exists
- Does nothing in dry-run mode
Inputs (globals):
MANIFEST_TMP
FLAG_DRYRUN
Returns
0 always
Usage
_manifest_discard
Notes:
- Safe to call multiple times
