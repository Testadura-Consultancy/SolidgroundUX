Allowed side effect:
- On source, the module may register groups and items with sgnd-console.
Example:
sgnd_console_register_group "system" "System tools" "General system operations"
sgnd_console_register_item \
"sys-status" \
"system" \
"System status" \
"sys_status" \
"Show system status" \
