# Changelog

## 1.5.0 - First Fully Documented Release

Released: June 2026

The second public release of SolidGroundUX and the first release accompanied by a complete documentation set, GitHub Pages site, installation packages, manifests, and release tooling.

SolidGroundUX is a framework for building structured Bash applications. It provides reusable services for bootstrap and runtime initialization, command-line argument processing, configuration management, persistent state variables, user interaction, terminal UI helpers, logging, datatable processing, documentation generation, deployment tooling, and modular console applications.

This release represents the current state of the framework after approximately six months of development and serves as the foundation for future releases.

### Highlights

* Complete generated HTML documentation
* GitHub Pages documentation site
* Framework bootstrap architecture
* Configuration and state management
* Command-line argument framework
* Terminal UI and dialog helpers
* Datatable processing library
* Deployment and release tooling
* Modular console infrastructure
* Project templates and SDK tooling
