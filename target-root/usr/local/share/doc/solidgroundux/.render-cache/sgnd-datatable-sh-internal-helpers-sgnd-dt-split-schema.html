sgnd_dt_split_schema
Purpose:
Split a schema string into positional column names.
Arguments:
$1 SCHEMA
Outputs:
Writes column names into global helper array SGND_DT_SPLIT.
Returns:
0 on success.
Usage:
sgnd_dt_split_schema "quot;$SCHEMA"quot;
Examples:
printf '#39;%s\n'#39; "quot;${SGND_DT_SPLIT[1]}"quot; # name
