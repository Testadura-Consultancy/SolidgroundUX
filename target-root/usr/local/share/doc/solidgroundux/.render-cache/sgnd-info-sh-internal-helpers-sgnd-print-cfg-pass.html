_sgnd_print_cfg_pass
Purpose:
Print configuration values for selected scopes from a named spec array.
Arguments:
$1 Header text.
$2 Spec array name.
$@ Accepted scopes after the first two arguments.
Inputs (globals):
_section_indent
_items_indent
Returns:
0 always.
Usage:
_sgnd_print_cfg_pass "quot;System globals"quot; SGND_FRAMEWORK_GLOBALS system both
Notes:
- Requires bash 4.3+ (nameref).
