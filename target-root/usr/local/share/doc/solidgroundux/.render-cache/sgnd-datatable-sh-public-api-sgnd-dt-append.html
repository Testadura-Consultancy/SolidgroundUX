sgnd_dt_append
Purpose:
Build and append a row from positional values.
Arguments:
$1 SCHEMA
$2 TABLE_ARRAY_NAME
$3+ VALUES
Returns:
0 on success.
1 on invalid values or width mismatch.
Usage:
sgnd_dt_append "quot;$SCHEMA"quot; ARRAY VALUE1 VALUE2 ...
Examples:
sgnd_dt_append "quot;$MY_SCHEMA"quot; MY_ROWS "quot;2"quot; "quot;Config"quot; "quot;Settings module"quot;
