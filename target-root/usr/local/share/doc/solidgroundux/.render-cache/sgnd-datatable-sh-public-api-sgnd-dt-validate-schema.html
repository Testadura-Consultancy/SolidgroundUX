sgnd_dt_validate_schema
Purpose:
Validate a schema string.
Rules:
- schema must not be empty
- each column name must be non-empty
- duplicate column names are not allowed
Arguments:
$1 SCHEMA
Returns:
0 if schema is valid.
1 if schema is invalid.
Usage:
sgnd_dt_validate_schema "quot;$SCHEMA"quot;
Examples:
if ! sgnd_dt_validate_schema "quot;$MY_SCHEMA"quot;; then
sayfail "quot;Invalid schema"quot;
fi
