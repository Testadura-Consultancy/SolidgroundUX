Console registration
Allowed side effect:
- On source, the module may register groups and items with sgnd-console.
Example:
sgnd_console_register_group "quot;system"quot; "quot;System tools"quot; "quot;General system operations"quot;
sgnd_console_register_item \
"quot;sys-status"quot; \
"quot;system"quot; \
"quot;System status"quot; \
"quot;sys_status"quot; \
"quot;Show system status"quot; \
