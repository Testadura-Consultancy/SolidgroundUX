Inputs (globals):
SGND_COMMON_LIB
SGND_CORE_LIBS
Behavior:
- Rebase directories.
- Sources each library from: $SGND_COMMON_LIB/gt;.
Side effects:
- Defines functions/variables provided by core libraries.
Returns:
Propagates any sourcing failures to the caller.
_source_corelibs
