sgnd_dt_make_row
Purpose:
Build a row string from positional field values.
Arguments:
$1 SCHEMA
$2+ VALUES matching schema order
Outputs:
Prints the constructed row string to stdout.
Returns:
0 on success.
1 on wrong field count or invalid field value.
Usage:
sgnd_dt_make_row "quot;$SCHEMA"quot; VALUE1 VALUE2 ...
Examples:
row="quot;$(sgnd_dt_make_row "quot;$MY_SCHEMA"quot; "quot;1"quot; "quot;Tools"quot; "quot;Utility module"quot;)"quot;
