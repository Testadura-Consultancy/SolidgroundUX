Behavior:
- Removes ESC[...gt; sequences from the supplied text.
- Intended primarily for SGR styling sequences used by the framework.
Arguments:
$1 TEXT
Text that may contain ANSI escape sequences.
Output:
Prints the sanitized string to stdout.
Returns:
0 always.
Usage:
sgnd_strip_ansi "quot;$text"quot;
Examples:
plain="quot;$(sgnd_strip_ansi "quot;$colored_text"quot;)"quot;
sgnd_strip_ansi
Purpose:
Strip ANSI CSI escape sequences from a string.
