sgnd_sgr
Purpose:
Build one canonical ANSI SGR escape sequence from mixed inputs.
Behavior:
- Accepts numeric SGR params, numeric param lists, and full ESC[...m sequences.
- Extracts and normalizes parameters into one combined SGR escape.
- Ignores empty or unsupported inputs.
- Returns RESET when no valid parts are supplied.
Arguments:
$@ PARTS
Mixed SGR parts such as:
- numeric params: 1
- param lists: "quot;1,4"quot; or "quot;1;4"quot;
- full escapes: $'#39;\e[38;5;46m'#39;
Inputs (globals):
RESET
Output:
Prints one SGR prefix to stdout.
Returns:
0 always.
Usage:
sgnd_sgr "quot;$WHITE"quot; "quot;"quot; "quot;$FX_BOLD"quot;
Examples:
clr="quot;$(sgnd_sgr "quot;$GREEN"quot; "quot;"quot; "quot;$FX_BOLD"quot;)"quot;
printf '#39;%sHello%s\n'#39; "quot;$(sgnd_sgr 1 4)"quot; "quot;$RESET"quot;
