sgnd_dt_get
Purpose:
Get one cell value from a table by row index and column name.
Arguments:
$1 SCHEMA
$2 TABLE_ARRAY_NAME
$3 ROW_INDEX
$4 COLUMN_NAME
Outputs:
Prints the field value to stdout.
Returns:
0 on success.
1 on invalid row index or invalid column.
Usage:
sgnd_dt_get "quot;$SCHEMA"quot; ARRAY INDEX COLUMN
Examples:
name="quot;$(sgnd_dt_get "quot;$MY_SCHEMA"quot; MY_ROWS 0 name)"quot;
