sgnd_bg
Purpose:
Convenience wrapper to build a background-color SGR prefix.
Arguments:
$1 BG
Background color escape or SGR fragment.
$@ FX
Optional effect fragments.
Output:
Prints an SGR prefix to stdout.
Returns:
0 always.
Usage:
sgnd_bg "quot;$BG_BLUE"quot;
Examples:
printf '#39;%sText%s\n'#39; "quot;$(sgnd_bg "quot;$BG_RED"quot; "quot;$FX_BOLD"quot;)"quot; "quot;$RESET"quot;
