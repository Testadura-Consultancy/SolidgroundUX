sgnd_print_license
Purpose:
Print the framework license text, including acceptance status.
Inputs (globals):
SGND_DOCS_DIR
SGND_LICENSE_FILE
SGND_LICENSE_ACCEPTED
TUI_INVALID
TUI_VALID
RESET
Outputs:
Prints formatted license text when the file is readable.
Returns:
0 always.
Usage:
sgnd_print_license
