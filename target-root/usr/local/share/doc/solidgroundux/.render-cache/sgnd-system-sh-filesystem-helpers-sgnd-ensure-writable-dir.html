sgnd_ensure_writable_dir
Purpose:
Ensure a directory exists and is suitable for user-scoped writing.
Arguments:
$1 DIR
Inputs (globals):
SUDO_USER
Returns:
0 on success.
2 if DIR is missing or empty.
3 if mkdir -p fails.
4 if the directory still does not exist afterward.
Usage:
sgnd_ensure_writable_dir DIR
