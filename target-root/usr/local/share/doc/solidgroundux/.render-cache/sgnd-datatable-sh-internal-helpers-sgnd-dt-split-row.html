sgnd_dt_split_row
Purpose:
Split a row string into positional field values.
Behavior:
- Preserves empty fields, including trailing empty values.
- Guarantees column count consistency with schema-based rows.
Arguments:
$1 ROW
Outputs:
Writes field values into global helper array SGND_DT_SPLIT.
Returns:
0 on success.
Usage:
sgnd_dt_split_row "quot;$ROW"quot;
Examples:
printf '#39;%s\n'#39; "quot;${SGND_DT_SPLIT[2]}"quot; # Utility
printf '#39;%s\n'#39; "quot;${SGND_DT_SPLIT[2]}"quot; # (empty string)
