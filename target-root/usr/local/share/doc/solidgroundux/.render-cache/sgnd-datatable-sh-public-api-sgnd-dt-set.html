sgnd_dt_set
Purpose:
Set one cell value in a table by row index and column name.
Arguments:
$1 SCHEMA
$2 TABLE_ARRAY_NAME
$3 ROW_INDEX
$4 COLUMN_NAME
$5 VALUE
Returns:
0 on success.
1 on invalid row index, invalid column, or invalid value.
Usage:
sgnd_dt_set "quot;$SCHEMA"quot; ARRAY INDEX COLUMN VALUE
Examples:
sgnd_dt_set "quot;$MY_SCHEMA"quot; MY_ROWS 0 name "quot;Updated"quot;
