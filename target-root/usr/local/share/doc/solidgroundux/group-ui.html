Modules and function glossaries

Modules
1 UI Ask — Provide interactive prompting and lightweight dialog utilities
  • ask — Prompt for interactive input from the controlling terminal (/dev/tty), [public]
  • _ask_build_prompt — Build a readline-safe prompt with visible-width padding and optional colors. [private]
  • _ask_choice_is_valid — Validate a single choice against a ask_choose-style choices list. [private]
  • ask_choose — Prompt for a constrained user choice using standard line input. [public]
  • ask_choose_immediate — Prompt for a constrained user choice using immediate-capable TTY input. [public]
  • ask_decision — Prompt for a constrained symbolic decision and store the canonical result. [public]
  • _ask_decision_display — Build a compact display string from a decision specification. [private]
  • _ask_decision_expand_choices — Expand a symbolic decision specification into canonical choices and aliases. [private]
  • _ask_decision_map_dlg_rc — Map ask_dlg_autocontinue return codes to a canonical decision when possible. [private]
  • _ask_decision_normalize — Normalize a typed decision value to its canonical token. [private]
  • ask_dlg_autocontinue — Show a lightweight timed auto-continue prompt on /dev/tty. [public]
  • _ask_expand_choices — Expand a comma-separated choice specification into individual allowed values. [private]
  • _ask_is_ident — Test whether a value is a valid shell variable identifier. [private]
  • _ask_parse_fieldspec — Parse a simple field-spec line into reusable globals for ask_prompt_form(). [private]
  • ask_prompt_form — Prompt for a list of variables described by field-spec lines. [public]
  • _ask_validate — Run a validation callback when provided. [private]
2 UI Dialogs — Provide dialog-style user interaction helpers
  • _dlg_keymap — Build a human-readable key legend string for the current dialog state. [private]
  • sgnd_ask_decision — Prompt for a constrained symbolic decision and normalize the result. [public]
  • sgnd_decision_display_choices — Build a compact display label for a decision specification. [public]
  • sgnd_decision_expand_choices — Expand a symbolic decision specification into canonical choices and aliases. [public]
  • sgnd_decision_from_dialog_rc — Translate a sgnd_dlg_autocontinue() return code into a canonical decision token. [public]
  • sgnd_decision_normalize — Normalize a typed value to its canonical decision token. [public]
  • sgnd_dlg_autocontinue — Render a timed dialog on /dev/tty with a countdown and simple key-driven actions. [public]
  • sgnd_prompt_fromlist — Prompt for a list of state or configuration entries described by state-spec lines. [public]

4 UI Messaging — Provide standardized console messaging helpers
  • say — Emit a typed message with formatting, colorization, and optional logging. [public]
  • say_caller — Resolve the originating caller location for logging purposes. [public]
  • saycancel — Emit a CNCL message. [public]
  • saydebug — Emit a DEBUG message with optional caller context. [public]
  • sayend — Emit an END message. [public]
  • sayfail — Emit a FAIL message. [public]
  • sayinfo — Emit an INFO message. [public]
  • sayok — Emit an OK message. [public]
  • sayprogress — Render single-line progress output on stderr and update it in place. [public]
  • sayprogress_done — Finalize progress output by printing a newline. [public]
  • say_should_print_console — Determine whether a message type should be printed to the console [public]
  • saystart — Emit a STRT message. [public]
  • saywarning — Emit a WARN message. [public]
  • say_write_log — Append a formatted log record to the logfile with optional rotation. [public]
  • _sgnd_logfile — Resolve the effective logfile path based on configured priorities. [private]
  • _sgnd_rotate_logs — Perform size-based rotation of a logfile. [private]
5 UI Rendering — Provide terminal rendering utilities for styled console output
  • confirm — Compatibility override for legacy yes/no confirmation prompts. [public]
  • sgnd_bg — Convenience wrapper to build a background-color SGR prefix. [public]
  • sgnd_color_samples — Print a demo table of available palette colors and common effects. [public]
  • sgnd_fg — Convenience wrapper to build a foreground-color SGR prefix. [public]
  • sgnd_module_show_metadata — Display metadata for a library module initialized via sgnd_module_init_metadata. [public]
  • sgnd_print — Print formatted text with padding, justification, and optional wrapping. [public]
  • sgnd_print_cell — Print a fixed-width cell that may contain ANSI styling sequences. [public]
  • sgnd_print_file — Print a text file to stdout, paging when interactive. [public]
  • sgnd_print_fill — Print one line with left and right content separated by a fill region. [public]
  • sgnd_print_labeledvalue — Print one aligned "quot;Label : Value"quot; line with optional styling. [public]
  • sgnd_print_sectionheader — Print a divider line with optional title text. [public]
  • sgnd_print_single — Render exactly one formatted line within a fixed width. [public]
  • sgnd_print_titlebar — Print a framed title bar with left/right header text and optional subtitle. [public]
  • sgnd_runmode_color — Return the ANSI prefix associated with the current run mode. [public]
  • sgnd_sample_row — Render one sample row for one or more named palette variables. [public]
  • sgnd_sgr — Build one canonical ANSI SGR escape sequence from mixed inputs. [public]
  • sgnd_strip_ansi — Strip ANSI CSI escape sequences from a string. [public]
  • sgnd_style_samples — Print a demo of semantic message and UI style variables. [public]
  • _sgnd_ui_resolve_theme_file — Resolve a palette or style specification into a readable .sh file path. [private]
  • sgnd_ui_set_default_theme — Convenience wrapper to load the default UI palette and style. [public]
  • sgnd_ui_set_theme — Load a UI palette and style into the current shell. [public]
  • sgnd_update_runmode — Update the styled global RUN_MODE indicator from the current dry-run state. [public]
  • sgnd_visible_len — Return the visible character length of a string after stripping ANSI. [public]
