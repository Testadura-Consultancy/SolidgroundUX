Bootstrap

Modules and function glossaries

Modules
1 Bootstrap Core — Initialize the SolidgroundUX framework and load core modules
  • _init_bootstrap — Initialize the early bootstrap environment before core library loading. [private]
  • _parse_bootstrap_args — Parse framework-level bootstrap switches before script/builtin parsing. [private]
  • sgnd_bootstrap — Initialize the Testadura framework runtime environment. [public]
  • sgnd_check_license — Verify that the current framework license has been accepted. [public]
  • sgnd_disable_save_state — Disable automatic state persistence. [public]
  • sgnd_enable_save_state — Enable automatic state persistence. [public]
  • sgnd_load_ui_style — Load the active UI palette and style definitions. [public]
  • sgnd_module_init_metadata — Define module-scoped metadata variables for a sourced library and [public]
  • sgnd_on_exit_add — Register a new EXIT handler to be executed by the EXIT dispatcher. [public]
  • sgnd_on_exit_install — Install the framework EXIT dispatcher (trap) exactly once per process. [public]
  • _sgnd_on_exit_run — Execute registered EXIT handlers (LIFO) while preserving the original exit code. [private]
  • sgnd_parse_statespec — Parse a pipe-delimited state specification into component fields. [public]
  • sgnd_save_state — Persist selected state variables to storage (as configured by SGND_STATE_VARIABLES). [public]
  • _sgnd_save_state_dispatch — Save state on EXIT only for clean exits. [private]
  • sgnd_script_init_metadata — Initialize executable metadata from structured header comments. [public]
  • _source_corelibs — Source core framework libraries in the defined order (SGND_CORE_LIBS). [private]
  • _source_usinglibs — Source optional script-requested libraries from SGND_COMMON_LIB. [private]
2 Bootstrap Environment — Initialize framework environment variables and directory structure
  • _build_framework_dirs — Rebuild the canonical list of framework directories from current path globals. [private]
  • sgnd_apply_defaults — Apply default values for bootstrap globals if currently unset. [public]
  • sgnd_defaults_reset — Reset framework-global configuration variables to their default state. [public]
  • sgnd_ensure_dirs — Ensure that framework directories exist and have appropriate ownership. [public]
  • sgnd_rebase_directories — Recompute framework, user, and script-scoped paths from current root settings. [public]
  • sgnd_rebase_framework_cfg_paths — Derive framework-global cfg file paths from current cfg directories. [public]
