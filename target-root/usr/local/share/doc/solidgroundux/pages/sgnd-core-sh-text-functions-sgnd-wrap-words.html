sgnd_wrap_words
Purpose:
Wrap a text string to a fixed width on word boundaries.
Arguments:
--width N
--text STR
Returns:
0 on success, including empty text.
2 on invalid arguments.
Usage:
sgnd_wrap_words --width N --text STR
