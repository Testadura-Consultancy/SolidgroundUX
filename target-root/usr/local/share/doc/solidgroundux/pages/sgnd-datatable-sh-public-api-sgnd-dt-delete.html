sgnd_dt_delete
Purpose:
Delete one row from a table array by index.
Arguments:
$1 TABLE_ARRAY_NAME
$2 ROW_INDEX
Returns:
0 on success.
1 on invalid row index.
Usage:
sgnd_dt_delete TABLE_ARRAY_NAME ROW_INDEX
