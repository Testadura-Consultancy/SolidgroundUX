sayfail
Purpose:
Emit a FAIL message.
Behavior:
- Delegates to say FAIL.
Returns:
0 always.
Usage:
sayfail "quot;Cannot create directory: $dir"quot;
Examples:
sayfail "quot;Bootstrap failed"quot;
