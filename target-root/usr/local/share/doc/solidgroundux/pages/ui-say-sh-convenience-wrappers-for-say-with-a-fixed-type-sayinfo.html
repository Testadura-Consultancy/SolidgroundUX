sayinfo
Purpose:
Emit an INFO message.
Behavior:
- Delegates to say INFO.
- # - Always returns 0, even when suppressed.
Inputs (globals):
SGND_LOG_LEVEL
Returns:
0 always.
Usage:
sayinfo "Loading optional libraries..."
Examples:
sayinfo "Using default configuration"
