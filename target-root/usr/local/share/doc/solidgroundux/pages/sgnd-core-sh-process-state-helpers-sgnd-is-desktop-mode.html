sgnd_is_desktop_mode
Purpose:
Detect whether a graphical desktop environment is available.
Behavior:
- Detects X11 or Wayland session
- Does not guarantee UI interactivity (separate concern)
Returns:
0 if desktop environment is available
1 otherwise
