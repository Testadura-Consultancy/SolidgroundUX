sgnd_default
Purpose:
Assign a default value to a variable when it is unset or empty.
Arguments:
$1 VAR_NAME
$2 DEFAULT
Side effects:
- Sets the target variable in the current shell.
Returns:
0 always.
Usage:
sgnd_default VAR_NAME DEFAULT
