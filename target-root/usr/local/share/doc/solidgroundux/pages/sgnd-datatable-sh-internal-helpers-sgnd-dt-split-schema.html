sgnd_dt_split_schema
Purpose:
Split a schema string into positional column names.
Arguments:
$1 SCHEMA
Outputs:
Writes column names into global helper array SGND_DT_SPLIT.
Returns:
0 on success.
Usage:
sgnd_dt_split_schema "$SCHEMA"
Examples:
printf '%s\n' "${SGND_DT_SPLIT[1]}" # name
