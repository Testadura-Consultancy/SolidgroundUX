sgnd_dt_insert
Purpose:
Append a row string to a table array.
Arguments:
$1 SCHEMA
$2 TABLE_ARRAY_NAME
$3 ROW
Returns:
0 on success.
1 on invalid schema or row width mismatch.
Usage:
sgnd_dt_insert "quot;$SCHEMA"quot; ARRAY_NAME ROW
