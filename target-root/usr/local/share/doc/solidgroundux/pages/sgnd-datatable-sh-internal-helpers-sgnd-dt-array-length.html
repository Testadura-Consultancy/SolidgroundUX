sgnd_dt_array_length
Purpose:
Return the length of an indexed array by name.
Arguments:
$1 ARRAY_NAME
Outputs:
Prints array length to stdout.
Returns:
0 on success.
Usage:
sgnd_dt_array_length ARRAY_NAME
Examples:
count="$(sgnd_dt_array_length MY_ROWS)"
