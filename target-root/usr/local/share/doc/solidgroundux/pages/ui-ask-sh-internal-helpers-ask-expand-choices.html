_ask_expand_choices
Purpose:
Expand a comma-separated choice specification into individual allowed values.
Arguments:
$1 Choice spec string. Supports:
- Literal tokens: "quot;dev,acc,prod"quot;
- Simple ranges: "quot;A-Z"quot;, "quot;0-9"quot; (single ASCII [[:alnum:]] endpoints)
- Optional whitespace around tokens: "quot;A-Z, 1-3, foo"quot;
Output:
Prints one expanded value per line.
Returns:
0 always.
