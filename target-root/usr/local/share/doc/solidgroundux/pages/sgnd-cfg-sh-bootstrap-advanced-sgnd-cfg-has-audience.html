sgnd_cfg_has_audience
Purpose:
Test whether a cfg spec array contains entries for a requested audience.
Behavior:
- Scans the supplied spec array.
- Matches entries marked for the requested audience.
- Treats "quot;both"quot; entries as matching either "quot;system"quot; or "quot;user"quot;.
Arguments:
$1 SPEC_ARRAY_NAME
Name of the specs array variable.
$2 AUDIENCE
Requested audience: "quot;system"quot; or "quot;user"quot;.
Returns:
0 if at least one matching spec exists.
1 otherwise.
Usage:
sgnd_cfg_has_audience SPEC_ARRAY_NAME system
Examples:
if sgnd_cfg_has_audience SGND_CFG_SPECS "quot;user"quot;; then
printf '#39;user cfg supported\n'#39;
fi
sgnd_cfg_has_audience SGND_CFG_SPECS "quot;system"quot;
