sgnd_dt_row_count
Purpose:
Return the number of rows in a table array.
Arguments:
$1 TABLE_ARRAY_NAME
Outputs:
Prints row count to stdout.
Returns:
0 on success.
Usage:
sgnd_dt_row_count TABLE_ARRAY_NAME
