_sgnd_ensure_linebreak
Purpose:
Ensure progress output is terminated before normal output continues.
Behavior:
- Prints a newline if a progress line is still active.
- Resets the pending linebreak flag.
Inputs (globals):
SGND_LINEBREAK_PENDING
Outputs (globals):
SGND_LINEBREAK_PENDING
