sgnd_dt_row_get
Purpose:
Get the value of one column from a row string.
Arguments:
$1 SCHEMA
$2 ROW
$3 COLUMN_NAME
Outputs:
Prints the field value to stdout.
Returns:
0 on success.
1 if the column is not found.
Usage:
sgnd_dt_row_get "$SCHEMA" "$ROW" COLUMN_NAME
