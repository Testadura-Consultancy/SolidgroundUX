sgnd_caller_id
Purpose:
Build a compact caller identifier string for diagnostics.
Arguments:
$1 DEPTH
Optional stack depth. Default: 1
Output:
Prints the caller identifier to stdout.
Returns:
0 always.
Usage:
sgnd_caller_id [DEPTH]
