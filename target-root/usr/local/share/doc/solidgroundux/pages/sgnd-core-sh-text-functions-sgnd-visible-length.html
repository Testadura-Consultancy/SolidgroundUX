sgnd_visible_length
Purpose:
Measure the character length of text after stripping ANSI escape sequences.
Arguments:
$1 TEXT
Output:
Prints the stripped character count to stdout.
Returns:
0 always.
Usage:
sgnd_visible_length TEXT
