_sgnd_cfg_write_template_header
Purpose:
Write a standard auto-generated header for a cfg template file.
Arguments:
$1 DOMAIN
Logical config domain name.
$2 AUDIENCE
Config audience: system or user.
Output:
Prints header text to stdout.
Returns:
0 always.
