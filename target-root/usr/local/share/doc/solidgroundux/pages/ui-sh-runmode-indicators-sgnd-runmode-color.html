sgnd_runmode_color
Purpose:
Return the ANSI prefix associated with the current run mode.
Inputs (globals):
FLAG_DRYRUN
TUI_DRYRUN
TUI_COMMIT
Output:
Prints the active run-mode color prefix to stdout.
Returns:
0 always.
Usage:
sgnd_runmode_color
Examples:
printf '%s%s%s\n' "$(sgnd_runmode_color)" "COMMIT" "$RESET"
