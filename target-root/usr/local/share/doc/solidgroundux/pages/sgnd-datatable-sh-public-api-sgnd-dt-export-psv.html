sgnd_dt_export_psv
Purpose:
Export a datatable array to a pipe-separated value file.
Arguments:
$1 SCHEMA
$2 TABLE_ARRAY_NAME
$3 OUTPUT_FILE
Returns:
0 on success.
1 on invalid input or write failure.
