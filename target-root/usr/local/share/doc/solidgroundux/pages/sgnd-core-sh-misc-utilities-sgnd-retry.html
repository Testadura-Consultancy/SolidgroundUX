sgnd_retry
Purpose:
Retry a command up to N times with a delay between attempts.
Arguments:
$1 ATTEMPTS
$2 DELAY_SECONDS
$@ COMMAND
Returns:
0 if the command succeeds within the retry budget.
1 if all attempts fail.
2 on invalid arguments.
Usage:
sgnd_retry ATTEMPTS DELAY COMMAND [ARG ...]
