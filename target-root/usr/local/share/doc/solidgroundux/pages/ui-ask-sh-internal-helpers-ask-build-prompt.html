_ask_build_prompt
Purpose:
Build a readline-safe prompt with visible-width padding and optional colors.
Arguments:
$1 LABEL
$2 LABELWIDTH
$3 PAD
$4 LABELCLR
$5 INPUTCLR
$6 COLORIZE
Output:
Writes the composed prompt string to stdout.
Returns:
0 always.
