saydebug
Purpose:
Emit a DEBUG message with caller context.
Behavior:
- Only active when SGND_LOG_LEVEL=debug.
- Appends caller file, function, and line.
- Delegates to say DEBUG.
Usage:
saydebug "quot;Value = $x"quot;
Examples:
saydebug "quot;Entering function"quot;
