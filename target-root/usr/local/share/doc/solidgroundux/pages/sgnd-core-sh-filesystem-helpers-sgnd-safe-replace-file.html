sgnd_safe_replace_file
Purpose:
Replace a destination file with a source file while preserving mode bits.
Arguments:
$1 SRC
$2 DST
Returns:
0 on success; 1 on failure.
Usage:
sgnd_safe_replace_file SRC DST
