sgnd_dt_column_count
Purpose:
Return the number of columns in a schema.
Arguments:
$1 SCHEMA
Outputs:
Prints column count to stdout.
Returns:
0 on success.
Usage:
sgnd_dt_column_count "$SCHEMA"
