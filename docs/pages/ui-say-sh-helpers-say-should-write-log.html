Determine whether a message should be written to the logfile
SolidgroundUX / ui-say.sh / Helpers / _say_should_write_log
Purpose
Apply the file log level independently from console visibility.
Arguments
$1 TYPE - Message type.
Inputs (globals):
SGND_FILE_LOG_LEVEL
Returns
0 when the message should be logged.
1 when the message should be suppressed.
