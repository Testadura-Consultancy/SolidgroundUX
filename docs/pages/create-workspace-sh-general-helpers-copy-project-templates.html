Copy project templates into the workspace
SolidgroundUX / create-workspace.sh / General helpers / _copy_project_templates
Purpose
Copy the appropriate template file(s) for the selected project components.
Behavior
- Copies exe-template when FLAG_EXE=1
- Copies lib-template when FLAG_LIB=1
- Copies mod-template when FLAG_MOD=1
- Uses component-specific output filenames
Inputs (globals):
PROJECT_NAME
PROJECT_FOLDER
FLAG_EXE
FLAG_LIB
FLAG_MOD
SGND_COMMON_LIB
Returns
0 on success
1 on failure
